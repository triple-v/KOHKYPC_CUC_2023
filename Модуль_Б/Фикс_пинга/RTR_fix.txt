После запуска или перезагрузки машины пишем следующее:



en    //  Вход в привилегированный режим



conf t    // Входим в настройки конфигурации

ip route 0.0.0.0 0.0.0.0 10.0.137.12



router eigrp 6500   // Входим в настройку сетевого адаптера

network 10.0.137.0 0.0.0.255

network 192.168.0.0 0.0.0.255

exit    // Выходим из настроек сетевого адаптера



reload    // Перезагружаем ВМ, на все вопросы в консоли пишем Y и жмём Enter
